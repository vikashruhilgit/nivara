# InvestIQ Mobile

Expo + React Native app for InvestIQ.

## Setup

```bash
cd mobile
npm install
npx expo start
```

## Stack
- Expo SDK 52 (New Architecture enabled)
- Expo Router v4 (file-based routing)
- expo-secure-store (bearer token storage — TechSpec v1.3)
- expo-notifications (push notifications)

## Directory Layout
- `app/` — expo-router screens (file-based routes)
- `assets/` — icons and splash (add before first build)
